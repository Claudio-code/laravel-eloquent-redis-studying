</p>

### How use?

- Build project
```
    make build
```

- Start project
```
    make start
```

- Run all Tests

```
    make run-all-tests
```

- Run lint
````
    make fix-all
````


### Technologies used this project

- PHP 8
- Laravel
- Redis
- Docker
- Mysql
- Bash
